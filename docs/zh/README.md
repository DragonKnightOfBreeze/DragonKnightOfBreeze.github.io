# 风之遗迹的大图书馆

传说，在风之一族世代守卫的古代遗迹中，坐落着一座纯白无瑕，仿佛不属于这个世界的大图书馆。  
而这座大图书馆的管理员，则是一名穿着象征灵使身份的深色风衣和三角帽，拥有白色长发和赤红瞳孔的风之子少女。
